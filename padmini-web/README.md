## 🛠️ ඇප් එක වැඩ කරන්නේ නැතිනම් (Troubleshooting)
යම් හෙයකින් ඇප් එකේ ප්‍රශ්නයක් ඇති වුවහොත් මෙම පියවර අනුගමනය කරන්න:

1. **API Keys පරීක්ෂාව:** `.env` ගොනුවේ අගයන් වල අගට හෝ මුලට හිස්තැන් (Spaces) නොමැති බව තහවුරු කරගන්න.
2. **Cloudflare Variables:** Cloudflare Dashboard එකේ `Settings -> Environment variables` වෙත ගොස් සියලුම Keys ඇතුළත් කර ඇති බව බලන්න.
3. **Redeploy:** Keys වෙනස් කළ පසු අනිවාර්යයෙන්ම Cloudflare හි "Retry Deployment" ලබා දෙන්න.
4. **Console Errors:** Browser එකේ F12 ඔබා `invalid-api-key` වැනි පණිවිඩයක් තිබේදැයි බලන්න.
